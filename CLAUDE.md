# DontStopCRM

CRM в стиле десктопного Telegram + Kommo. AI-администратор ногтевого салона: ведёт календарь, записывает брони, переносит, ведёт досье клиентов, ставит себе задачи на следующий контакт.

## Экраны (MVP)

- **Дашборд** — главный экран, статистика, ближайшие задачи
- **Лиды** — канбан 4 колонки (Новый → Контакт → Квалифицирован → Выигран)
- **Чаты** — мессенджер с лидами, подключение агентов
- **Календарь** — записи, дела, привязка к лидам

## Стек

- **Backend:** FastAPI + PostgreSQL + SQLAlchemy + Alembic
- **Frontend:** Vue 3 + TypeScript + Tailwind CSS
- **Библиотеки:** vuedraggable (канбан), FullCalendar (календарь), Chart.js или ApexCharts (дашборд)
